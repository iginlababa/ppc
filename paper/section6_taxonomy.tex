\subsection{Failure Mode Categories}

Every root cause is attributed to one of four categories:

\begin{enumerate}
    \item \textbf{Compiler Backend Failure} --- poor code generation: missed optimizations,
          bad loop unrolling, suboptimal register allocation. Detected via PTX/GCN inspection.
    \item \textbf{Runtime Coordination Overhead} --- extra synchronization, task scheduling
          costs, dispatch overhead. Detected via \texttt{nsys} timeline.
    \item \textbf{Memory Model Mismatch} --- abstraction forces non-optimal layout or extra
          indirection. Detected via L2 cache hit rate and memory access pattern analysis.
    \item \textbf{API Limitation} --- abstraction cannot express a required optimization
          (e.g., shared memory tiling). Detected via source code analysis.
\end{enumerate}
